    		•  Choose this if you want to use HL7 communication.
  • Microsoft SQL Server 2008 Management Studio Express
    		•  Choose this if you want to maintain the database.

    		•  Choose this if you want to use GB18030 encoding.
  • Microsoft Access Database Engine 2010 Redistributable
    		•  Choose this if you want to read CSV file.
